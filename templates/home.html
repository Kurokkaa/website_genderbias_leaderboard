Welcome to Masculead, an interactive leaderboard

This website allows you to evaluate gender biases of language models for French texts, using the GenderGap and/or GenderShift metrics. The computed results can then be added to MascuLead, our leaderboard!

1st Step: Generate some cover letters

Go to the GitHub repository and use the generation script with the autoregressive language model of your choice.

2nd Step: Automatically annotate the generations

You can either go to the Upload page, put your generations file and select "no" for the annotation question. After some time, you will get the bias metrics. Otherwise, you can run the gender detection script yourself, and upload the annotated generation file, selecting "yes" to the annotation question to get the metrics.

Upload your csv

3rd Step: Compare your results

Now you can check your results in the leaderboard. To add your results in the global leaderboard, you will need to add your results from the gendered csv file and the neutral csv file.
